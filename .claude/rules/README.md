# Path-scoped rules

These files capture conventions that apply to specific directories. They are
referenced from `CLAUDE.md` and read on demand by agents working in a matching
path. Keep each file short — anything that should always apply belongs in
`CLAUDE.md`.

| File | Applies to |
|---|---|
| `data-json.md` | `data/*.json` |
| `app-routes.md` | `app/**` |
| `lib-tests.md` | `lib/*.ts` and `lib/*.test.ts` |
| `components.md` | `components/**` |
| `scripts.md` | `scripts/**` |
